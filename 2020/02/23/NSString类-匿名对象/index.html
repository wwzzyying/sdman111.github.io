NSString类 & 匿名对象

2020-02-23

NSString类

NSString是Foundation框架的一个类，是用来存储OC字符串的，而OC字符串本质上用 NSString对象 保存；

完整的创建NSString对象方式：

  • @”jack”是一个OC字符串，本质上是NSString对象；NSString *str = @"jack";创建了一个指向OC字符串@”jack”的对象，对象的地址保存在str指针变量中。
  • %p打印指针变量的值；
  • %@打印指针变量指向的对象。

NSString最常用的方法

类方法

  1. + (nullable instancetype)stringWithUTF8String:(const char *)nullTerminatedCString;
把C的字符串转换为OC的字符串对象，返回值为当前类的对象。

+ (instancetype)stringWithFormat:(NSString *)format, ... NS_FORMAT_FUNCTION(1,2);
一个或多个字符串对象拼接起来。

length方法，返回值为NSUInteger(实际上是 unsigned long)
用于求字符串长度。中英文字符串都可以，可以中英混合。
为什么中文6个和英文6个长度都显示6呢？
    -因为NSString的length方法返回一个UTF-16编码单元（码元）的个数，即大部分字符都以固定长度的字节 (2字节) 储存。调用时看看使用了多少个UTF-16编码的单元就得到长度了。

得到字符串指定下标的字符
- (unichar)characterAtIndex:(NSUInteger)index;
返回值是unichar，其实就是unsigned short，占两个字节；
调用：
    unichar ch = [str characterAtIndex:3];
输出：
    NSLog(@"ch=%c",ch);小写c，输出char类型-英文
    NSLog(@"ch=%C",ch);大写C，输出unichar类型-可以显示中文

判断两个字符串的内容是否相同，不要用==去判断
判断方式：
- (BOOL)isEqualToString:(NSString *)aString;
调用：
    [str isEqualToString:str2];

比较字符串大小
- (NSComparisonResult)compare:(NSString *)string;
可以用int接收，因为返回值是一个枚举;
-1是小于，0是等于，1是大于。

[Person new]->_name = @"xxx";
[Person new]->_age = 18;
[[Person new] sayHi]调用sayHi方法，sayHi方法本质上输出对象属性，但此时输出的分别是null和0，因为每一次new都是一个新的对象，属性不是此前赋值的@"xxx"和18.

匿名对象用处

  • 某个对象的成员只使用一次，然后这个对象就不用了；
  • 如果方法的参数是一个对象，且专门给某个方法传递，且调用者不再使用这个对象，就可以传一个匿名对象。

例子：上帝杀人，创建一个人的对象作为一个参数传入上帝杀人的方法，杀完之后这个被杀的对象就不用了。
