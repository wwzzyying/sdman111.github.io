蓝桥杯_方格填数问题

2019-12-30

问题描述

如下的10个格子

+–+–+–+
| | | |
+–+–+–+–+
| | | | |
+–+–+–+–+
| | | |
+–+–+–+

（如果显示有问题，也可以参看【图1.jpg】）

填入0~9的数字。要求：连续的两个数字不能相邻。
（左右、上下、对角都算相邻）

一共有多少种可能的填数方案？

做的题太少，一开始没想到深搜一个一个找然后再回溯，这里递上DFS的过程

judge函数检测那里，一开始看到别人的代码是不检测下一行的我就纳闷了，不是要检测八个方向是否冲突吗，然后我用了这个，好像也没错，只是没必要。

然后后面看到解释，因为后面四个方向如果按照从左到右还是上到下的顺序都是初值为1111或者回溯后初值变1111的，所以不用检测后面四个方向的数。

然后就是递归方向那里

看了别人的才知道是从左到右从上到下。
