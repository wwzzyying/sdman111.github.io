LeetCode 1、两数之和总结

2019-11-26

只想到暴力法了…

学习疑问1：函数中的指针参数int* returnSize干啥的？

找到答案：如果没有returnSize，main函数就不知道该从twoSum函数返回的地址处取几个int数据 ，所以，没有给指针变量returnSize赋值，编译会不通过。

学习疑问2：runtime error: load of null pointer of type ‘int’

找到解释：因为这题返回的是指针类型，而该指针变量a指向的是局部变量，当跳出函数时，该变量作用域失效，就会发生错误。这里换成在循环内部动态分配内存，而分配到堆内存里的数据的生命周期为从malloc分配开始到free释放结束，只要未执行到主函数的free()方法，就不会报错。

反正就是让该变量作用域尽可能长，详情看[踩坑记录] runtime error: load of null pointer of type ‘const int’ (leetcode)

学习疑问3：malloc函数返回类型问题

一开始没有在malloc前面加(int *)导致报错，后面发现malloc的函数原型是void *malloc(unsigned int size)，也就是说malloc函数返回值是未确定类型指针，需要强制转换类型。

学习疑问4：关于指针参数

这里的returnSize指针保存的是和实参一样的地址，可以通过修改指针所指向内存单元的值来改变实参所指向内存单元的值，因为二者所指向的地址都是一样的。但需要先进行解引用，才可以改变地址的内容。
