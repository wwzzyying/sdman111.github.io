在App Inventor内调用接口

2019-11-19

选了可视化移动编程这节课之后没少发呆，终于在期末课程设计前学了点东西了，就是如何调用一个外部接口，实现远程访问别人公开出来的你需要的信息。

这里的接口指的是API： Application Programming Interface ，通过调用别人已经规范定义好的代码来实现功能。

调用接口的前提：给程序装上HTTP客户端组件，用于程序的通信连接。

在这里我示范如何调用一个实时汇率的接口，因为期末要做一个课程设计，感觉也就接口有趣点，因为我觉得AI的定制化程度不高，只能在已知的我所学会的组件中找最有创造性的组件了。
